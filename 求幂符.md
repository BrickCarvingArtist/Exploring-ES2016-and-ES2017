# 3. 取幂符 (**)

> 取幂符 (**) 是一个由 `Rick Waldron` 提出的 `ECMAScript` 提议。

## 3.1 概述
``` JavaScript
> 6 ** 2

36
```
3.2 用于求幂的中缀运算符

** is an infix operator for exponentiation:

x ** y
produces the same result as

Math.pow(x, y)
Examples:

let squared = 3 ** 2; // 9

let num = 3;
num **= 2;
console.log(num); // 9
