## 3. 取幂符 (**)

> 取幂符 (`**`) 是一个由 `Rick Waldron` 提出的 `ECMAScript` 提案。 

### 3.1 概述
```JavaScript
> 6 ** 2

36
```
### 3.2 用于求幂的插入运算符

`**` 是一个 求幂的插入运算符： 
```JavaScript
x ** y
```
与下面的表达式产生相同的结果： 
```JavaScript
Math.pow(x, y)
```
例子：
```
let squared = 3 ** 2; // 9
let num = 3;
num **= 2;
console.log(num); // 9
```
